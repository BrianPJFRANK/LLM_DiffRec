Running Command Example:

1. Orginal DiffRec
bash run_semantic_npu.sh amazon-instruments 0.00025 0.00008 128 "[150, 400]" 10 x0 4 0.0012 0.006 0.025 0 1 original_standard 1 0 0 original 1024 256

2. Semantic Model
bash run_semantic_npu.sh amazon-instruments 0.00025 0.0001 128 "[150, 400]" 10 x0 8 0.0012 0.006 0.025 0 1 semantic_steps8 1 0 1 semantic 1024 256

3. Dual-Stream Semantic Model
bash run_semantic_npu.sh amazon-instruments 0.0001 0.00008 256 "[150, 400]" 10 x0 4 0.0012 0.006 0.025 0 1 dual_lowLR 1 0 1 dual 1024 256

4. FiLM Model
bash run_semantic_npu.sh amazon-instruments 0.00025 0.00008 128 "[200, 600]" 10 x0 10 0.0012 0.006 0.025 0 1 film_steps10 1 0 1 film 1024 256

5. FiLM-dot Model (cold-start)
bash run_semantic_npu.sh amazon-instruments_coldstart 0.001 0.0005 128 "[512, 1024]" 10 x0 8 0.0012 0.006 0.025 0 1 cold_filmdot_highLR_extreme_step8 1 0 1 film_dot 1024 256

6. Inference in standard dataset
python inference_semantic.py --dataset amazon-instruments --data_path ../datasets/ --model_path ./saved_models_semantic/<YourModelName.pth> --use_semantic --model_type film --batch_size 128 --semantic_dim 1024 --cuda --gpu 0

7. Inference in coldstart dataset
python inference_semantic.py --dataset amazon-instruments --data_path ../datasets/ --model_path ./saved_models_semantic/<YourModelName.pth> --use_semantic --model_type film --batch_size 128 --semantic_dim 1024 --cuda --gpu 0
